# Sistema de Gestión – Obligatorio Programación 3 (ORT)

Proyecto académico desarrollado como parte de la materia Programación 3
de la carrera Analista en Tecnologías de la Información (Universidad ORT Uruguay).

## Descripción
Sistema de gestión desarrollado en C#, enfocado en la correcta aplicación
de Programación Orientada a Objetos, lógica de negocio y validaciones.
El proyecto modela entidades del dominio y aplica reglas de negocio
según los requerimientos del obligatorio.

## Tecnologías utilizadas
- C#
- .NET
- SQL
- GitHub (control de versiones)

## Funcionalidades principales
- Modelado de entidades del dominio
- Validaciones de reglas de negocio
- Persistencia de datos
- Estructura de código clara y modular

## Contexto académico
Proyecto realizado con fines académicos.
El foco está puesto en la calidad del código, la lógica de negocio
y el diseño orientado a objetos.
